# Voice Casting Studio Published Shared Forms

Place public form payload JSON files here.

Example:

```text
public/shared-forms/voice-casting.json
```

The matching public URL is:

```text
https://umbrellaparade.github.io/Voice-Casting-Studio/#/r/voice-casting
```

Only publish form-definition payloads created by the app's "公開JSON" buttons.
Do not publish response JSON, audio files, private notes, or credentials here.
